# UDP

---

## 什么是UDP

一种用户数据报协议，又称用户数据报文协议

一个简单的面向数据报的传输层协议，正式规范为RFC 768

用户数据协议、非连接协议



## 为什么不可靠

它一旦把应用程序发给网络层的数据发送出去，就不保留数据备份

UDP在IP数据报的头部仅仅加入了复用和数据校验（字段）

发送端生产数据，接收端从网络中抓取数据

结构简单、无校验、速度快、容易丢包、可广播



## UDP能做什么

DNS、TFTP、SNMP

视频、音频、普通数据（无关紧要数据）



## UDP包最大长度

16位->2字节 存储长度信息

2^16-1 = 64k-1 = 65536-1 = 65535

自身协议占用：32+32 = 64位 = 8字节

65535 - 8 = 65507 byte



## UDP单播、广播、多播

单播：点对点

多播：组播，给一个组发信息

广播：给所有设备发送信息
